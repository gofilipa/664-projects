This project uses the Runescape API to pull the prices of items and
display them in a series of charts. It runs automatically every day at
3pm, scraping the current prices and generating new charts, in a
workflow configured by Github Actions.

Tools used:
- `Runscape API`
- `pandas` python library
- `requests` python library
- Github Actions workflow
